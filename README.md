# SmartHome Dashboard

Giao diện web để theo dõi và điều khiển các thiết bị thông minh trong nhà, sử dụng Firebase Realtime Database.


![z6924534006431_bf41552b46ad4b3ee6483e925168003e](https://github.com/user-attachments/assets/17ce5e82-f4a7-4eb3-a544-a169133f79d5)

## Tính năng
- Theo dõi nhiệt độ, độ ẩm, và cường độ ánh sáng trong thời gian thực.
- Điều khiển các thiết bị (đèn, quạt, điều hòa) qua các nút bật/tắt.
- Chế độ ban đêm (night mode) để thay đổi giao diện.
- Thiết kế responsive, hỗ trợ cả desktop và mobile.
